# Stellar Sea

Babylon.js + TypeScript prototype for `Stellar Sea`.

## Current goal

Phase 1 proves the visual direction first:

- pixel-style 3D planet
- one visible auto-harvester beacon
- orbit camera for inspecting landing spots

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Next step

Implement the actual harvesting loop:

- surface resource nodes
- placeable harvester device
- auto-production state and HUD feedback
